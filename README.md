NBTEditor
=========

An in-game NBT editor for [Bukkit].

The NBTEditor is built and tested for CraftBukkit and Spigot ([SipgotMC BuildTools][BuildTools]).

More information on the [NBTEditor BukkitDev page][NBTEditor].
